# Documentation Index

Use this folder as the operational reference for the project.

## Files

- `docs/RUNBOOK.md`
  - Step-by-step command workflow for training, plotting, and evaluation.
- `docs/TROUBLESHOOTING.md`
  - Common Windows + Ray RLlib errors and fixes/workarounds.
- `docs/EXPERIMENT_PLAN.md`
  - Milestones, evaluation gates, experiment matrix, and weekly execution plan.

## Recommended Order

1. Read `docs/RUNBOOK.md` and run a quick training + quick eval.
2. If anything fails, use `docs/TROUBLESHOOTING.md`.
3. Track weekly progress and milestones from `docs/EXPERIMENT_PLAN.md`.
